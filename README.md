backboneMvc-requirejs

项目结构流程如图所示，
---

<pre>
  backboneMvc  增加控制器逻辑, 效果类似cakephp。 在前端通过requirejs路由动态加载匹配的控制器文件执行对应的action。
  
  开发模式下 requirejs按需加载， 根据路由动态加载匹配的controller  view  model文件。开发调试更直观。
  
  生产环境 打包成压缩文件走预加载。
</pre>
